#Express Server

Express Server is a easy to use node.js webserver based on express.js. It came with an integrated overview of your REST API. 

##Integration
Create folowing folder structure in your project root:

`
projectRoot
--------------------------
+-- server
	+-- env
		+-- development.js
		+-- production.js
	+-- database.js
	+-- express.js
	+-- init.js
+-- routes
	+-- myRoutes.js
	+-- moreRoutes.js
`
